# Empire State Walkers

Professional dog walking in Manhattan.

## What It Is

A minimalist website for a dog walking service. No frameworks. No build process. Just HTML, CSS, and JavaScript.

## Design Philosophy

Brutalist-minimalist aesthetic. Black and white. Monospace typography. No gradients, shadows, or unnecessary decoration. Function over form.

## Features

- Service listings
- Contact form
- User authentication (localStorage-based)
- Customer dashboard
- Responsive design

## Stack

- HTML5
- CSS3 (custom, no frameworks)
- Vanilla JavaScript
- Space Mono (Google Fonts)

## Structure

```
index.html    — Everything
styles.css    — Styling
script.js     — Interactions
```

## Run It

Open `index.html` in a browser. Or:

```bash
python -m http.server 8000
```
